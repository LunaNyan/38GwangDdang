?. game 클래스
    현재 진행중인 게임의 진행 과정을 적재한 클래스로 되어있다.

?-1. game 클래스 내 함수
    stake (int) : 게임의 판돈
    call_cost (int) : 콜비용 씨발ㄹ련아 왜안만들었노
    turn (int) : 게임의 턴 카운트
    win (class) : 게임의 승리자(user 클래스가 적재됨)가 적재되며, 게임 종료 시 직전까지는 None 상태를 유지한다.

?. player 함수
    플레이어, NPC 등 섯다 진행이 가능한 유저 인스턴스의 데이터를 적재한 클래스로 되어있다.

?.1. player 클래스 내 함수
    money (int) : 유저 인스턴스가 현재 갖고 있는 돈
    dead (bool) : 유저가 다이했는지를 마킹하는 함수

?. 패 함수
    패의 자료형은 2개의 int를 가진 list로 정의된다.
    0번 항목은 월수, 1번 항목은 패의 종류(끗)을 의미한다.

?-1. 패의 종류(끗)
    0 : 기타
    1 : 광
    2 : 열끗
